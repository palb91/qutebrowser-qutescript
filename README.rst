cycle-owa
=========

::

    Cycle through user folders in office365.

    arguments:
        next    cycle from top to bottom (default)
        prev    cycle from bottom to top

    binding exemple:
        :bind « spawn -u cycle_owa
        :bind « spawn -u cycle_owa next
        :bind » spawn -u cycle_owa prev
